## üîß Troubleshooting Guide

### Issue 1: CapCut Crashes or Freezes

**Symptoms**: App becomes unresponsive, timeline lags

**Solutions**:
1. **Save frequently**: File ‚Üí Save Project (Ctrl+S)
2. **Reduce preview quality**: Settings ‚Üí Preview Quality ‚Üí Low
3. **Close other apps**: Free up RAM
4. **Restart CapCut**: Close and reopen
5. **Update CapCut**: Check for latest version

### Issue 2: Audio Sync Issues

**Symptoms**: BGM doesn't match video action

**Solutions**:
1. **Manual adjustment**: Click BGM track, drag to offset
2. **Use markers**: Add markers at key beats, align clips
3. **Trim BGM start**: Cut intro if climax is too late
4. **Speed adjustment**: Slightly speed up/slow down clips (98-102%)

### Issue 3: Export Quality Degradation

**Symptoms**: Final video looks blurry or pixelated

**Solutions**:
1. **Increase bitrate**: Export settings ‚Üí Bitrate ‚Üí 15-20 Mbps
2. **Use "Best" quality**: Export settings ‚Üí Quality ‚Üí Best
3. **Check source quality**: Ensure input clips are high-res
4. **Disable hardware acceleration**: Settings ‚Üí Export ‚Üí Software encoding

### Issue 4: Watermark Appears

**Symptoms**: CapCut logo visible in corner

**Solutions**:
1. **CapCut Pro trial**: Sign up for free trial (7-30 days)
2. **Crop video**: Slightly crop to remove watermark area
3. **Alternative editor**: Export without watermark using DaVinci Resolve
4. **Mobile vs PC**: Try PC version (often no watermark)

### Issue 5: Subtitle Positioning Issues

**Symptoms**: Subtitles cut off on mobile, not in safe area

**Solutions**:
1. **Raise position**: Move subtitles higher (bottom 1/4 instead of 1/3)
2. **Reduce size**: Smaller font (60-80pt instead of 80-100pt)
3. **Test on mobile**: Preview on phone before finalizing
4. **Use safe area guides**: Enable safe area overlay in CapCut

### Issue 6: Transition Timing Too Fast/Slow

**Symptoms**: Transitions feel jarring or sluggish

**Solutions**:
1. **Adjust duration**: Click transition ‚Üí Duration ‚Üí 0.3-0.5s
2. **Change transition type**: Try different effect
3. **Remove transition**: Delete and use straight cut
4. **Preview multiple times**: Watch at full speed to judge

### Issue 7: File Size Too Large (>100MB)

**Symptoms**: Export exceeds 100MB limit

**Solutions**:
1. **Reduce bitrate**: Export settings ‚Üí Bitrate ‚Üí 8-10 Mbps
2. **Compress video**: Use HandBrake or online compressor
3. **Trim duration**: Cut to 15-20 seconds instead of 30
4. **Lower resolution**: Export at 720p, then upscale (not recommended)
